Portfolio Pessoal - Full Stack
Este é o meu Portfólio Profissional, desenvolvido para centralizar meus projetos, experiências e habilidades técnicas. A aplicação foi construída utilizando Django para uma gestão de conteúdo dinâmica e Docker para garantir um ambiente de implantação moderno e isolado.

Propósito do Projeto
O objetivo deste projeto é ir além de um site estático, demonstrando habilidades em:

Backend robusto: Gerenciamento de projetos e categorias via banco de dados.

Performance: Otimização de carregamento de imagens de projetos (Media files).

Interface: Design responsivo e focado na experiência do usuário (UX).

Stack Tecnológica
Backend: Django 5.2+

Banco de Dados: Sqlite

Frontend: HTML5, CSS3, JavaScript

Servidor de Arquivos: Pillow (para processamento de imagens)

Funcionalidades
Showcase de Projetos: Galeria dinâmica com filtros por tecnologias (tags).

Blog Integrado: Espaço para artigos técnicos e atualizações.

Currículo Online: Seção de experiência profissional e formação acadêmica.

Formulário de Contato: Sistema de mensageria com armazenamento no banco.

Admin Customizado: Painel para gerenciar todo o conteúdo sem tocar no código.
